ErrorHandler¶

class ampl::ErrorHandler¶

A basic interface for AMPL error handlers.

If an application needs to implement customised error handling, it must implement this interface and then register an instance with the AMPL API using the AMPL::setErrorHandler method. The underlying AMPL interpreter will then report all errors and warnings through this interface.
